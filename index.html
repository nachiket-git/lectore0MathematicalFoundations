1. Singular Value Decomposition (SVD)

Singular Value Decomposition (SVD) is a factorization of a matrix A into three components: A = U Σ V^T, where:

  • U: Orthogonal matrix containing the left singular vectors.
  • Σ: Diagonal matrix with singular values.
  • V^T: Transpose of an orthogonal matrix containing the right singular vectors.
SVD is widely used in dimensionality reduction, noise reduction in images, and recommender systems.

Example Problem

Decompose the matrix:

import numpy as np
# Define matrix A
A = np.array([[3, 1], [1, 3]])

# Perform SVD
U, S, VT = np.linalg.svd(A)
print("Matrix U (Left Singular Vectors):", U)
print("Singular Values (Diagonal of Σ):", S)
print("Matrix V^T (Right Singular Vectors Transposed):", VT)
